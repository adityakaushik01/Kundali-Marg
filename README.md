# 🧿 Kundali Marg – Vedic Astrology Birth Chart Generator

Kundali Marg is a modern **Vedic Astrology web application** that generates an accurate **Janam Kundali (Birth Chart)** using astronomical calculations powered by the **Swiss Ephemeris library**.

The project combines **modern frontend technologies with traditional Jyotish calculations** to deliver a fast, accurate, and visually intuitive astrology experience.

# ✨ Features

### 🔮 Kundali Generation
Generate a complete Vedic birth chart using:

- Name
- Date of Birth
- Time of Birth
- Birth Location (Latitude & Longitude)

---

### 🪐 Accurate Astronomical Calculations
Powered by **Swiss Ephemeris**, ensuring high-precision planetary calculations.

Includes:

- Lagna (Ascendant)
- Planetary Positions
- Rashi (Zodiac Signs)
- Nakshatra & Pada
- Retrograde Detection
- House Placement
- Rasi Chart Generation

---

### 📊 Detailed Chart Data
The application calculates and displays:

# 🛠 Tech Stack

## Frontend

- React 19
- Vite
- Tailwind CSS
- Material UI
- React Router
- Day.js

---

## Backend

- Node.js
- Express.js
- Swiss Ephemeris (`swisseph`)
- Express Rate Limiting
- CORS

---

## Astronomy Engine

The backend uses **Swiss Ephemeris**, one of the most accurate astronomical libraries used in astrology software worldwide.

It calculates:

- Planetary longitudes
- Ayanamsa (Lahiri)
- Ascendant
- House cusps
- Nakshatra & Padas
